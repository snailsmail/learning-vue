问题：
1.vue的格式要求很严格
    注解单独占一行，且//后需要空格
    出现()则前后都需要空格
    声明的数据必需要用单引号声明
    name属性值也必需要用单引号

2.组件的通讯是单向的，父组件可以向子组件传值，子组件使用props:[“num”],接收值。
  子组件不能直接向父组件传值，可以通过this.$emit(‘bb’)触发父组件的方法改变父组件的值。
